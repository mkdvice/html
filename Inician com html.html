A WEB

A web, é composta por clientes, internet e servidores. Essa comunicação é feita através do chamado HTTP - HyperText Trnsfer Protocol(protocolo de transferência de hipertexto)

HTML

As páginas da Web, em sua maioria são criadas em HTML(HyperText Markup Lenguage). Essa linguagem de marcação de hipertexto, é uma das linguagens que os navegadores interpretam, nos mostrando algo mais "clean", mostrando só o que é necessario ao usuário

No HTML temos as chamadas tags responsaveis por modificar o texto, para que fique do modo que o programador deseja, modificando os textos para negrito, itálico, adicionar links e imagens, quebras de linhas e outras modificações.
